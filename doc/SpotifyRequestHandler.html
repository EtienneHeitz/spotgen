clientsCredentialsFlow(clientId, clientSecret, grantTypeopt) → {Promise|JSON}

Authenticate with the Clients Credentials Flow.

Note: this authentication method only works if the script is run from the command line. It does not work when run from a browser, because Spotify's authentication server rejects cross-site requests. In that case, authenticate with the Implicit Grant Flow instead and pass the received access token to this class.
